CI5.HOST // AUDIT_NODE

Isolated forensic environment for Raspberry Pi 5 / OpenWrt.

To audit a community cork before installation:

curl ci5.host/audit | sh -s [cork-name]

This will:

  • Snapshot your current filesystem.
  • Run the Cork in an air-gapped Docker bridge.
  • Diff all changes to your Host Overlay.
  • Track unauthorized "phone-home" attempts.
